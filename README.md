# LearnAI - AI Powered Smart Learning Companion

LearnAI is an AI-powered educational platform designed to provide personalized learning experiences for students. The platform offers interactive courses, AI-assisted tutoring, progress tracking, and adaptive learning recommendations.

## Features

* Personalized Learning Paths
* AI Tutor Chatbot
* Course Recommendations
* Progress Tracking Dashboard
* Responsive Design
* Modern User Interface

## Technologies Used

* HTML5
* CSS3
* JavaScript
* GitHub Pages
